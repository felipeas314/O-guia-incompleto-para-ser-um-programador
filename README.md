# O Guia Incompleto para Ser um Programador

> *"Todo expert já foi um iniciante."* — Helen Hayes

---

## Sobre o Projeto

Este é um projeto **100% gratuito e open source** com um único objetivo: **ajudar pessoas a aprenderem a programar do zero**.

- Não vendemos cursos
- Não temos mensalidade
- Não pedimos seu email para liberar conteúdo

O objetivo aqui é **educação**, não dinheiro.

Se você nunca escreveu uma linha de código na vida, este guia foi feito para você.

---

## Por Que "Incompleto"?

Porque a programação é um universo infinito. Nenhum guia consegue cobrir tudo — e quem disser o contrário está mentindo.

Este guia te dá uma **base sólida** para você começar sua jornada. Com ela, você terá autonomia para continuar aprendendo por conta própria, seja qual for o caminho que escolher seguir.

O nome também é um lembrete: **você nunca vai saber tudo**, e tudo bem. O importante é começar.

## Como Usar Este Guia

### 1. Siga a Ordem
Os capítulos foram pensados para serem lidos em sequência. Cada um constrói sobre o anterior. Não pule etapas!

### 2. Pratique Muito
Cada capítulo tem:
- **Explicação teórica** com exemplos práticos
- **Exercícios resolvidos** passo a passo
- **Lista de exercícios** para você praticar (15 questões por capítulo)
- **Respostas comentadas** para conferir

**Não pule os exercícios!** Programação se aprende programando, não lendo.

### 3. Digite o Código
Não copie e cole. Digite cada linha de código você mesmo. Parece bobeira, mas faz toda a diferença no aprendizado. Seu cérebro aprende diferente quando você digita.

### 4. Erre Bastante
Erros são seus amigos. Cada erro é uma oportunidade de aprender algo novo. Leia as mensagens de erro — elas geralmente explicam o problema.

### 5. Não Tenha Pressa
Programação leva tempo para aprender. Não se compare com outros. Foque no seu próprio progresso.

## Requisitos

**O que você precisa:**

- Paciência (programação leva tempo!)

**O que você NÃO precisa:**

## Linguagem Utilizada

Este guia usa **Python** para ensinar programação. Por quê?

| Vantagem | Descrição |
|----------|-----------|
| **Sintaxe simples** | Código limpo e fácil de ler |
| **Versátil** | Usado em web, dados, IA, automação... |
| **Popular** | Grande comunidade e muitos recursos |
| **Gratuito** | Totalmente open source |
| **Demanda** | Uma das linguagens mais requisitadas no mercado |

Python é uma excelente primeira linguagem. Depois de aprendê-la, migrar para outras linguagens fica muito mais fácil.

## Contribuindo

Encontrou um erro? Tem uma sugestão? Quer adicionar conteúdo? Contribuições são muito bem-vindas!

### Como contribuir:

1. Faça um **fork** do projeto
2. Crie uma **branch** para sua alteração (`git checkout -b minha-alteracao`)
3. Faça **commit** das mudanças (`git commit -m "Descrição da alteração"`)
4. Faça **push** para a branch (`git push origin minha-alteracao`)
5. Abra um **Pull Request**

### O que você pode contribuir:

- Correção de erros de português ou código
- Melhoria nas explicações
- Novos exercícios
- Tradução para outros idiomas
- Sugestões de conteúdo
